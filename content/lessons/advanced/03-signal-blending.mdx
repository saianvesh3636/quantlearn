# Blending Uncorrelated Signals

Combining multiple uncorrelated signals creates more robust strategies than any single signal alone.

<ConceptCard title="Diversification of Alpha" type="definition">
Just as you diversify assets, you should **diversify signals**. If you have three signals with Sharpe 0.5 and zero correlation, combining them can achieve Sharpe ~0.87. Diversification works on alpha too!
</ConceptCard>

## The Math of Combination

<FormulaDisplay
  formula="SR_{combined} = \sqrt{SR_1^2 + SR_2^2 + ... + SR_n^2}"
  explanation="Combined Sharpe ratio when signals are uncorrelated."
  variables={[
    { symbol: "SR_{combined}", meaning: "Sharpe ratio of combined strategy" },
    { symbol: "SR_i", meaning: "Sharpe ratio of individual signal i" },
    { symbol: "n", meaning: "Number of signals" }
  ]}
/>

<ConceptCard title="Why This Works" type="tip">
When signals are uncorrelated:
- They make mistakes at different times
- Errors partially cancel out
- Combined signal is more stable

This is the quant's free lunch - if you can find uncorrelated alpha sources.
</ConceptCard>

## Weighting Schemes

<ConceptCard title="How to Combine" type="example">
**Equal weight**: Simple, robust, no estimation error
**Sharpe-weighted**: Weight by historical Sharpe ratio
**Inverse-vol weighted**: Weight by 1/volatility
**Optimized**: Mean-variance optimal (but beware overfitting!)
</ConceptCard>

<FormulaDisplay
  formula="w_i = \frac{SR_i / \sigma_i}{\sum_j SR_j / \sigma_j}"
  explanation="Information-ratio weighted combination."
  variables={[
    { symbol: "w_i", meaning: "Weight on signal i" },
    { symbol: "SR_i", meaning: "Sharpe ratio of signal i" },
    { symbol: "\\sigma_i", meaning: "Volatility of signal i" }
  ]}
/>

<KeyTakeaway>
Combining uncorrelated signals improves risk-adjusted returns through **alpha diversification**. The combined Sharpe ratio can exceed any individual signal's Sharpe. Use correlation analysis to find diverse signals, then blend with simple weighting schemes to avoid overfitting.
</KeyTakeaway>
